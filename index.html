CODE SAMPLES
  • Relate.
    Relate is a platform for team bonding games that build on each other to support community-wide ideation and deliberation. As a Lead Developer for Relate in Summer 2020, I co-led a team of five developers to create a responsive UI, managed team tasks and Firebase backend, and coordinated with a design team and 2 other dev teams.
  • Causeway.
    Causeway allows anyone to work on small development roles that contribute to real-world projects and teach professional skills along the way. As Causeway's Lead Developer in June 2020 through March 2021, I designed and developed user onboarding for a key part of the platform, debugged and built on a large codebase, and helped conduct 3 rounds of user research. I led two teams of 3 developers to adapt to feedback and improve our user experience.
  • Escher Level Generator.
    (Public Source Code)
    I wrote this level generator in Python. It's designed to create a text file that represents a 3D level based on command line specifications, which can be imported into our Unity project. A C# script then uses the generated text file to construct the playable level.
  • In A Flash.
    (Public Source Code)
    In A Flash was a solo project in Unity and C#. I designed and implemented 2 AI opponents with varying levels of game awareness and difficulty. I also created 4 distinct unit types with upgrades, an interactive tutorial, a gameplay HUD and menus.
  • Sea Star Crossed Lovers.
    (Public Source Code)
    SSCL was created in Unity, in collaboration with 80+ developers at GDA Collab, UC Santa Cruz. I collaborated on obstacle design, and implemented a Pause Menu UI. I also implemented sound effects and music in Unity.
  • keyvalue.
    (Private for academic integrity, email bjpaulse@ucsc.edu for employer access)
    keyvalue is a C++ implementation of a dictionary interface for a balanced binary search tree, able to store and lookup information with O(log(n)) runtime. Created with pair programmer Jon Zivku.
  • yshell.
    (Private for academic integrity, email bjpaulse@ucsc.edu for employer access)
    yshell is a C++ implementation of a UNIX-style terminal, complete with a fully memory-based filesystem. Created with pair programmer Jon Zivku.
  • BigInt.
    (Private for academic integrity, email bjpaulse@ucsc.edu for employer access)
    BigInt is a C++ implementation of the dc calculator program. It can handle calculations with integers well beyond the normal memory range of signed longs. Created with pair programmer Jon Zivku.

    (Private for academic integrity, email bjpaulse@ucsc.edu for employer access)

  • FindPath.
    (Private for academic integrity, email bjpaulse@ucsc.edu for employer access)
    FindPath is a C implementation of Breadth First Search for Graphs. This algorithm finds the shortest path to any node.
  • BigInteger.
