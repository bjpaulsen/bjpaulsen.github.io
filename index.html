ABOUT ME

Hello, I’m Benjamin! I am a sophomore studying Computer Science, Game Design at the University of California, Santa Cruz.

Right now I'm working at Tech4Good Lab, creating interfaces for learning platforms that will change the way we teach STEM. I'm also a Learning Support Services tutor, helping my fellow students get into engineering and find community at our school. I'm interested in researching computer science, education, game design, and putting my skills to the test!

I’m interested in cooperating with other engineers to build our knowledge and learn to develop effective, inclusive teams. That’s why I became a tutor. I believe that if we truly listen to one another, and utilize our diverse set of experiences as students and engineers, we can create a welcoming and empowering place of learning.

I have experience serving the public, from children and teens, to adults and seniors. I have worked at a local library, helped organize and run two blood drives, fed our community at Food Not Bombs, and cared for animals at the Santa Cruz SPCA. For more on what I can contribute, be sure to read my resume. I look forward to working with you!

GAMES

  • Flock Together. A flying game about finding your chosen family. Soar over the watercolor landscape and flock with other birds. Created in collaboration with Zac Nizibian.
  • Pew Pew. An original Galaga-style arcade game. Written using Javascript OOP and p5js.
  • Flicky. A Star-Trek inspired, turn-based strategy game. Control a fleet of starships and defeat the enemy faction! Choose from four empires, multiple scenarios, and compete with 1-3 friends for control of the galaxy!
  • Froggo. Froggo is a simple, minimlist game inspired by the classic hit "Frogger". The initial release was written over the course of three days as part of an independent study of Javascript.
  • Dino. Inspired by Chrome's internet disconnect page, Dino is my rendition of the classic side-scroller jumping game, complete with a scoreboard so you can compete with friends.
  • Ping. A Pong clone that comes with an AI to play against -- or it can even play against itself!
  • Pew. To my surprise, I couldn't find the original Asteroids game anywhere on the internet -- so I made my own version!
  • The Frog. An original branching-story game about the weirdness, hilarity, and devastation of early romantic relationships.
  • Snake. The classic, with added game modes and features. Huge credit to Prashant Gupta, whose example and starter code helped start me on my journey into js game design.

PROGRAMS

  • keyvalue. (Link is private for academic integrity, email me for employer access) keyvalue is a C++ implementation of a dictionary interface for a balanced binary search tree, able to store and lookup information with O(log(n)) runtime. Created with Jon Zivku for CSE 111, Advanced Programming.
  • yshell. (Link is private for academic integrity, email me for employer access) yshell is a C++ implementation of a UNIX-style terminal, complete with a fully memory-based filesystem. Created with Jon Zivku for CSE 111, Advanced Programming.
  • BigInt. (Link is private for academic integrity, email me for employer access) BigInt is a C++ implementation of the dc calculator program. It can handle calculations with integers well beyond the normal memory range of signed longs. Created with Jon Zivku for CSE 111, Advanced Programming.
  • FindComponents. (Link is private for academic integrity, email me for employer access) FindPath is a C implementation of Depth First Search for Graphs. This algorithm finds each connected component of a graph, along with other useful structural data. Created for CSE 101, Algorithms and Abstract Data Types.
  • FindPath. (Link is private for academic integrity, email me for employer access) FindPath is a C implementation of Breadth First Search for Graphs. This algorithm finds the shortest path to any node. Created for CSE 101, Algorithms and Abstract Data Types.
  • BigInteger. (Link is private for academic integrity, email me for employer access) BigInteger is a C implementation of big integer arithmetic. This implementation is highly efficient and again limited only by memory space. Created for CSE 101, Algorithms and Abstract Data Types.
  • Sparse. (Link is private for academic integrity, email me for employer access) Sparse is a project in C which builds matrices from and input file and performs efficient matrix operations, especially on large, sparse matrices. Created for CSE 101, Algorithms and Abstract Data Types.
  • Lex. (Link is private for academic integrity, email me for employer access) Lex is a C implementation of insertion sort. It outputs the lines of a file in lexicographic order. Created for CSE 101, Algorithms and Abstract Data Types.
  • LoopPedal. LoopPedal is a live music performance tool that lets a solo artist become like an orchestra. Built at HackACM 2019 in under 12 hours by me, Ankur Mishra, Kai O'Brien, and Robin Rae. Won the hackathon beginner category.
  • FinalGrade. FinalGrade is a grade planner tool for students.
  • SlugBus++. A bus tracking app developed by Jorge Henriquez, John Hanly, Justyn Duthler, Jeremy Diamzon, and Case Delst at CruzHacks 2019. I later joined the team to improve on the map interface and help design a new bus stop UI.
